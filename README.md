# Majoo Test Case Web Dev

## Installation

```sh
git clone https://github.com/hamzahfauzy/aws-lex-chatbox
cd aws-lex-chatbox
composer install
cp .env.example .env
php artisan key:generate
```

Set AWS_ACCESS_KEY_ID and AWS_SECRET_ACCESS_KEY in .env file

## Run Serve
```sh
php artisan serve
```

## Accessing the application
```sh
http://127.0.0.1:8000
```
